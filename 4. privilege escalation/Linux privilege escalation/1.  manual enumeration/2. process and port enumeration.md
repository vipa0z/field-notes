 list processes with their  PID, PPID, UID, user, cmdline (forest shows parent-child)

```
# (look for cronjobs, script files (.sh), parent processes, bind ports
ps -ef --forest |less -S
```

```
ps -eo pid,ppid,user,uid,cmd --forest
```

 wide output with start time and more
```
ps -eo pid,ppid,user,stime,etime,pcpu,pmem,args --sort=-pcpu

# listing by user
ps -u <username> -o pid,ppid,uid,user,cmd
```

# ) Find binary path, cwd, open files for a PID
```
# binary path of process
readlink -f /proc/<PID>/exe

# current working directory of process
readlink -f /proc/<PID>/cwd

# command line (NUL-separated -> readable)
tr '\0' ' ' < /proc/<PID>/cmdline ; echo

# list open files (including sockets) held by process
sudo lsof -p <PID>

# list mapped libraries
cat /proc/<PID>/maps | less

```
### show listening TCP/UDP with PIDs and programs
```
ss -tulpen            

# alternative (netstat)
sudo netstat -tulpen

# list listening processes via lsof
sudo lsof -i -P -n | grep LISTEN

# show Unix domain sockets
ss -x -a -p

```
